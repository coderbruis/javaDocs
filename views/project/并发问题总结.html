并发问题总结

CoderBruis 2021-07-24 12:21:00 项目总结

在开发过程中，会有许多的并发问题没有注意到，包括并发类、并发集合工具类的使用等等，通过此篇文章来收集问题并总结（持续更新）。

# 1. putIfAbsent

对于Java中的Map，先判断key是否存在，不存在则立马设置新值的操作相信大家都不会陌生，代码如下：

    if (map.containsKey(key)) {
        map.put(key, map.get(key) + value);
    } else {
        map.put(key,value);
    }

但是在并发环境下，在判断完key是否存在之后再put，在put的时候可能其他线程就已经put完了，导致线程不安全，所以需要使用并发集合类的putIfAbsent操作来解决。

    if (map.containsKey(key)) {
        map.putIfAbsent(key, value);
    }
